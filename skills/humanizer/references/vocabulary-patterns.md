# 어휘 패턴 (14~20, 37~39)

LLM 학습 데이터의 빈출 어휘가 한국어 텍스트에 강하게 반영. 인간이 일상에서 거의 쓰지 않는 추상어·hype 어휘가 군집을 이룬다.

---

## 패턴 14: AI 유행어 과다 사용

- **검증**: 📊 경험적 (LLM 출력 통계)
- **심각도**: S1
- **버전**: v1.0.0

**현상**: 다음 어휘가 한 문서에 군집으로 출현:

`중요하다`, `핵심적`, `효과적`, `지속가능한`, `혁신적`, `포괄적`, `체계적`, `궁극적`, `근본적`, `필수적`, `결정적`

**검출 기준**:
- 한 문단에 위 어휘 2개 이상
- "~적인 X" 형태로 반복

**예시 (AI)**:
- "이는 혁신적이고 지속가능한 핵심 전략이다. 효과적인 실행이 결정적이다."

**교정**:
- "이건 새롭고 오래 갈 전략이다. 잘 실행하는 게 관건이다."
- 구체적 어휘로 대체 (혁신적 → 새로운, 효과적 → 잘 되는/유효한)

## 패턴 15: 불필요한 한자어 과다

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: 일상 표현 자리에 무거운 한자어 사용. "사용한다" 대신 "활용한다", "한다" 대신 "수행한다".

**검출 기준**:
- 동사 자리에 3~4음절 한자어 동사 반복
- 활용/수행/시행/실시/이행/추진/구축/도모/모색

**예시 (AI)**:
- "본 프로젝트는 새로운 방법론을 활용하여 시스템 구축을 추진한다."

**교정**:
- "이 프로젝트는 새 방법으로 시스템을 만든다."

## 패턴 16: 영어 표현 직역

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: "leverage" → "레버리지하다/활용하다", "ensure" → "보장하다", "facilitate" → "촉진하다", "enable" → "가능하게 하다".

**검출 기준**:
- "보장하다" "촉진하다" "가능하게 하다" 반복
- 영어 비즈니스 어휘의 1:1 대응

**예시 (AI)**:
- "이 도구는 협업을 촉진하고 생산성을 보장합니다."

**교정**:
- "이 도구로 협업이 잘 되고 생산성도 올라갑니다."

## 패턴 17: 복수형 '-들' 과다 사용

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: 한국어는 본래 복수 표지 생략. LLM은 영어 plural을 그대로 "-들"로 옮김.

**검출 기준**:
- "사용자들이" "기능들을" "방법들" 반복
- 문맥상 복수 표지가 불필요한데도 붙어 있음

**예시 (AI)**:
- "개발자들은 다양한 도구들을 활용해 기능들을 구현한다."

**교정**:
- "개발자는 여러 도구로 기능을 구현한다."

## 패턴 18: 대명사 과다 사용

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: "그것", "이것", "그들" 같은 대명사를 영어처럼 명시. 한국어는 생략이 자연.

**검출 기준**:
- 문장마다 대명사 등장
- 직전 명사를 가리키는데도 "그것은" 명시

**예시 (AI)**:
- "이 시스템은 빠르다. 그것은 안정적이기도 하다. 그것을 도입하면 효율이 높아진다."

**교정**:
- "이 시스템은 빠르고 안정적이다. 도입하면 효율이 올라간다."

---

## 패턴 19: 지시관형사 과다 ('이/그/저')

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: "이 방법", "그 결과", "이러한 접근"이 반복. 한국어는 맥락으로 충분할 때 생략.

**검출 기준**:
- 문단마다 "이/그/이러한/그러한" 출현
- "이러한 점에서" "그러한 의미에서" 메타 표현

**예시 (AI)**:
- "이러한 결과는 이 방법의 효과를 보여준다. 그러한 점에서 이 접근은 유효하다."

**교정**:
- "결과는 방법의 효과를 보여준다. 이 접근은 유효하다."

## 패턴 20: 주어 생략 미흡

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: 한국어는 주어 생략이 자연스러우나 LLM은 영어처럼 매 문장 주어 명시.

**검출 기준**:
- 모든 문장에 주어가 박혀 있음
- "우리는", "이 글은", "본 보고서는" 반복

**예시 (AI)**:
- "우리는 이 문제를 분석했다. 우리는 해결책을 제안한다. 우리는 결과를 검증할 것이다."

**교정**:
- "이 문제를 분석하고 해결책을 제안한다. 결과는 검증할 예정이다."

## 패턴 37: AI 결말 관용구

- **검증**: 📊 경험적
- **심각도**: S1
- **버전**: v1.5.0

**현상**: 글 마지막에 정해진 공식 문구로 마무리. "결론적으로", "요약하면", "이상으로", "앞으로의 발전이 기대된다", "더 나은 미래를 위해".

**검출 기준**:
- 마지막 문단이 메타 요약 + hype 결말
- "이러한 노력은 ~로 이어질 것이다"

**예시 (AI)**:
- "결론적으로, 이 접근법은 미래의 가능성을 열어줄 것이다."

**교정**:
- "이 접근법으로 다음 단계가 보인다."
- 결말 공식 자체를 제거하고 본문 마지막 문장으로 자연 종결

## 패턴 38: hype 어휘 군집

- **검증**: 📊 경험적
- **심각도**: S1
- **버전**: v1.5.0

**현상**: "최첨단", "획기적", "혁신적", "독보적", "차세대" 같은 마케팅 어휘가 한 문단에 몰림.

**검출 기준**:
- 문단당 hype 어휘 2개 이상
- 객관적 근거 없이 추상 칭찬

**예시 (AI)**:
- "이 차세대 솔루션은 혁신적이며 획기적인 변화를 이끈다."

**교정**:
- "이 솔루션은 X와 Y를 다르게 해서 [구체적 효과]를 낸다."

## 패턴 39: ~적 N 추상 체인

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.5.0

**현상**: "~적 [명사]" 구조가 사슬처럼 연결. "전략적 접근의 체계적 적용을 통한 효율적 운영의 지속가능한 실현".

**검출 기준**:
- 한 문장에 "~적" 3개 이상
- 명사 사이에 "~적"이 연쇄

**예시 (AI)**:
- "체계적 접근의 효율적 적용을 통한 전략적 성과의 지속가능한 창출이 목표이다."

**교정**:
- "체계 있게 접근해서 효율을 높이고 전략적 성과를 꾸준히 내는 것이 목표다."
- "~적" 사슬을 풀어 구체 동사로 전환
