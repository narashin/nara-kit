# 품사 다양성 패턴 (11~13)

KatFishNet 82.99% AUC. LLM 텍스트는 명사 비율이 비정상적으로 높고 동사/형용사가 빈약. 한국어는 본래 용언(verb/adjective) 중심 언어임에도 LLM은 영어식 명사 중심 구조를 옮김.

---

## 패턴 11: 명사 과다 사용 (명사화 구문)

- **검증**: ✅ 과학적 (KatFishNet, 82.99% AUC)
- **심각도**: S1
- **버전**: v1.0.0

**현상**: 동사를 명사화해서 "~의 진행" "~의 수행" "~의 도입"처럼 영어 "the X of Y" 구조를 옮김. 한국어 자연 화법은 "X를 한다/도입한다"로 동사 사용.

**검출 기준**:
- "~의 [명사화어]" 구문 빈도
- "진행", "수행", "도입", "활용", "구현", "적용"이 명사형으로 반복
- "X를 함으로써" "X의 결과로" 같은 무거운 구조

**예시 (AI)**:
- "프로젝트의 성공적인 완료를 위한 전략의 수립이 필요하다."
- "사용자 경험의 개선과 시스템 성능의 최적화를 동시에 달성한다."

**교정**:
- "프로젝트를 성공적으로 마치려면 전략을 세워야 한다."
- "사용자 경험을 개선하면서 시스템 성능도 최적화한다."

## 패턴 12: 동사/형용사 빈곤

- **검증**: ✅ 과학적 (KatFishNet)
- **심각도**: S2
- **버전**: v1.0.0

**현상**: 동사·형용사 사용 빈도가 낮고 같은 어휘 반복. 인간은 동의어 동사를 자연스럽게 섞음.

**검출 기준**:
- "~이다" "~있다" "~한다" 같은 일반 동사가 비정상적으로 반복
- 형용사 어휘 폭이 좁음 ("좋다/중요하다/효과적이다"만 반복)

**예시 (AI)**:
- "이 방법은 효과적이다. 시간 절약이 가능하다. 결과가 만족스럽다."

**교정**:
- "이 방법으로 시간을 아꼈고 결과도 마음에 든다."
- 동사 다양화: 절약하다 → 아끼다/줄이다/단축하다

## 패턴 13: 명사 나열식 표현

- **검증**: 📊 경험적
- **심각도**: S2
- **버전**: v1.0.0

**현상**: "A, B, C 등을 통한 D의 E" 같은 명사 나열 구조. 영어 noun phrase 직역.

**검출 기준**:
- 쉼표로 연결된 명사 3개 이상 + 조사 "을 통한"
- 동사 없이 명사만 연속

**예시 (AI)**:
- "데이터 수집, 분석, 시각화를 통한 인사이트의 도출이 핵심이다."

**교정**:
- "데이터를 모으고 분석한 다음 시각화해서 인사이트를 뽑는 게 핵심이다."
