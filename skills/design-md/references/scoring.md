# Completeness Score

## Criteria

| Criteria | COMPLETE | PARTIAL | MISSING |
|----------|----------|---------|---------|
| Colors | All project colors with hex + semantic role | Colors listed, roles partial | Template colors unchanged |
| Typography | Full hierarchy matching project fonts | Fonts replaced, hierarchy partial | Template fonts unchanged |
| Components | 5+ project components with actual values | 1-4 components | Template components unchanged |
| Layout | Spacing + grid + breakpoints from project | Spacing only | Template layout unchanged |
| Overview | Prose reflects actual project personality | Generic/template description remains | Section untouched |

## Score

**Score = COMPLETE count / 5**:

- **5/5 PRODUCTION** — "디자인 스펙 완전. AI 에이전트가 바로 사용 가능"
- **3-4/5 USABLE** — "주요 토큰 교체 완료. 나머지 보완 권장"
- **0-2/5 DRAFT** — "템플릿 구조만 있음. 프로젝트 토큰 교체 필요"
